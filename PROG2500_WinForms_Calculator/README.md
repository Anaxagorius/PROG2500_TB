
# PROG 2500 – Windows Forms Calculator

## Build & Run (Windows with .NET 6 SDK)

1. Open a Developer PowerShell/Command Prompt.
2. Navigate into `PROG2500_WinForms_Calculator`.
3. Run: `dotnet build`
4. Run: `dotnet run`

This project constructs the UI in code (no Designer file) to ensure a self-contained submission.
